# Learn Cosmology

This repository contains the scripts and notebooks about learning the usages of the commonly used packages in the cosmology research.

The packages (or tools) include:
- [Abacus](Abacus/README.md): A high-performance N-body simulation code, and its products and analysis tools.
- ClustatKit: would be an individual package for clustering measurements including 2PCF, power spectrum etc. (in progress)
